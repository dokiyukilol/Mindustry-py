# mindustry-python
## 文件说明

 - Snake_BP.msch 
由pyndustric-master编译成的贪吃蛇游戏的蓝图文件，可以直接在mindustry中使用。

 - snake.py
包含了一个简单的贪吃蛇游戏的实现。需要中型逻辑。需要通过pyndustric-master转为逻辑。

 - 逻辑_钍反应堆控制器.txt
用于控制反应堆，当温度过高时自动停机，需要最小逻辑。
Thorium reactor controller, used to regulate the reactor, automatically shuts down when the temperature is too high.
